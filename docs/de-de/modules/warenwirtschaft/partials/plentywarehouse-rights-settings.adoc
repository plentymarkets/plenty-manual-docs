[IMPORTANT]
.Notwendige Berechtigungen beachten
====
Die unten beschriebenen Einstellungen können standardmäßig nur von Benutzer:innen mit Rollen vom Typ *Administrator* vorgenommen werden. Sollen andere App-Benutzer:innen die Einstellungen sehen und bearbeiten können, müssen Administratoren entsprechende Rechte im plentymarkets Backend vergeben. Öffne dazu die entsprechende Rolle im Menü *Einrichtung » Kontoverwaltung » Rollen* und aktiviere im Bereich *plentyWarehouse* die gewünschten Rechte für die Option *Einstellungen*.
====
