[#git-repository-loeschen]
= Git-Repository löschen

Wenn du ein Git-Plugin nicht mehr benötigst, kannst du das zugehörige Repository löschen, um das Plugin vollständig aus deinem System zu entfernen. Damit du ein Git-Repository löschen kannst, musst du vorher <<plugin-aus-plugin-set-loeschen, das zugehörige Plugin aus allen Plugin-Sets löschen>>.
