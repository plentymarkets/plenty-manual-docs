= Marketplace-Plugins

Wenn ein Update für ein Plugin verfügbar ist, wird in der Spalte *Aktionen* die Schaltfläche *Aktualisieren* farbig gekennzeichnet.

include::partial$plugin-aktualisieren.adoc[]

Es ist bei Marketplace-Plugins nicht möglich, die Version auf eine ältere herunterzustufen. Wenn du eine ältere Version eines Plugins verwenden möchtest, musst du es xref:plugins:installierte-plugins-entfernen.adoc#plugin-aus-plugin-set-loeschen[löschen] und dann xref:plugins:hinzugefuegte-plugins-installieren.adoc[neu installieren].
