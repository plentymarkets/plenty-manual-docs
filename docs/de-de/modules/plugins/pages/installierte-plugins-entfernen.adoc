= Installierte Plugins entfernen
:author: team-plugin-core
:keywords: plugin, plugin, erweiterung, addon, git, github, gitlab, bitbucket, repository
:description: Wenn du ein Plugin nicht mehr nutzen möchtest, kannst du das Plugin entweder deaktivieren oder löschen.
:id: MSCTNZP

Es gibt verschiedene Möglichkeiten, installierte Plugins zu entfernen. Du kannst Plugins einzeln <<plugin-deaktivieren, deaktiveren>>, damit sie nicht mehr in deinem Webshop und Backend geladen werden oder sie vollständig <<plugin-aus-plugin-set-loeschen, aus einem Plugin-Set löschen>>. Wenn du alle Plugins eines Sets löschen möchtest, kannst du stattdessen <<plugin-set-loeschen, das Plugin-Set selbst löschen>>.

Du kannst Plugins, die du auf plentyMarketplace erworben hast, nicht vollständig aus deinem System entfernen. Handelt es sich allerdings um ein Git-Plugin, kannst du <<git-repository-loeschen, das zugehörige Repository löschen>>.
