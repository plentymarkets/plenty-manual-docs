[#git-plugins]
= Git-Plugins

Wenn du plentymarkets erweitern möchtest, aber kein Plugin mit der entsprechenden Funktionalität auf plentyMarketplace finden kannst, hast du die Möglichkeit, deinem System Plugins über Git hinzuzufügen. Manche externen Entwickler:innen bieten Plugins auf diesem Weg an. Wenn du weitere Informationen zu so verfügbaren Plugins erhalten möchtest, solltest du direkt mit diesen Entwicklern Kontakt aufnehmen. Nutze dafür entweder die link:https://marketplace.plentymarkets.com/partners[Partnerübersicht^], schreibe eine Nachricht in der Forumskategorie link:https://forum.plentymarkets.com/c/suche-biete/suche-biete-plugins[Suche/Biete^] oder erstelle eine link:https://marketplace.plentymarkets.com/plentyprojects[neue Projektanfrage^]{nbsp}icon:external-link[] auf plentyMarketplace.

[NOTE]
.Keine Prüfung durch plentymarkets
====
Wenn ein Plugin auf plentyMarketplace hochgeladen wird, wird es durch Mitarbeiter:innen von plentymarkets geprüft. Bei Plugins, die nicht über plentyMarketplace vertrieben werden, findet keine Prüfung statt. Das heißt, dass plentymarkets die Funktionalität des Plugins in diesen Fällen nicht sicherstellen kann.

Möchtest du ein Plugin von einem Drittanbieter aus einer anderen Quelle als plentyMarketplace in deinem System einbinden, empfehlen wir dir, dich datenschutzrechtlich abzusichern und den Funktionsumfang des Plugins nach dem Einbinden gründlich zu prüfen.
====
