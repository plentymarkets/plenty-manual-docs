[#kennzahl]
Kennzahl:: Eine Kennzahl gibt Aufschluss über betriebswirtschaftliche Zusammenhänge und stellt somit eine Entscheidungshilfe für unternehmerische Entscheidungen dar. In plentymarkets können Kennzahlen mithilfe vielfältiger Einstellungen im Modul <<business-entscheidungen/plenty-bi#, plentyBI>> angelegt und statistisch ausgewertet werden. +
* Weitere Informationen findest du auf der Handbuchseite <<business-entscheidungen/plenty-bi/kennzahlen#, Kennzahlen>>.
