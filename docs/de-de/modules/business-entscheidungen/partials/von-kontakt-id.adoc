=== Von Kontakt-ID

Filtert die ausgewählte Kennzahl und Dimension zusätzlich nach einer oder mehreren Kontakt-IDs. Zur Auswahl stehen dir für diesen Filter die in deinem System verfügbaren xref:crm:kontakt-suchen.adoc#[Kontakt-IDs]. Wähle einen der folgenden Vergleichsoperatoren aus und gib die gewünschten Kontakt-IDs kommasepariert in das Textfeld ein.
