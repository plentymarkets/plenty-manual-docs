[#netto-warenbestand]
Netto-Warenbestand:: Menge der Artikel, die zum Verkauf zur Verfügung steht. Wenn vom physischen Warenbestand alle Reservierungen abgezogen werden, ergibt sich der Netto-Warenbestand. +
