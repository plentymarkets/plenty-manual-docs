[#entnahme]
Entnahme:: Physische Reduzierung des Bargeldbestandes, um beispielsweise Beträge von der Kasse auf andere Konten zu transferieren. xref:pos:pos-kassenbenutzer.adoc#200[Entnahmen] werden in der plentymarkets App oder im Backend verbucht und können verschiedenen Buchungskonten zugewiesen werden.
