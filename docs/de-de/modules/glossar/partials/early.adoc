early:: Ein Entwicklungsstand von plentysystems. Funktionalität wird auf dieser Version veröffentlich, wenn auf <<#beta, beta>> keine Probleme festgestellt wurden. Wenn auch auf diesem Entwicklungsstand keine Probleme festgestellt werden, wird die Funktionalität auf <<#stable, stable>> veröffentlicht. +
