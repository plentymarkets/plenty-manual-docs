ifdef::import[]

Falls ja, dann gib den passenden Amazon-Marktplatz in die CSV-Datei ein.

*_Standardwert_*: `0`

[cols="1,1"]
|===
|Zulässige Importwerte in CSV-Datei |Optionen in der Dropdown-Liste im Backend

|`0`
|Nicht verwenden

|`1`
|AMAZON_EU

|`2`
|AMAZON_FE

|`3`
|AMAZON_NA

|`-1`
|FBA und SellerCentral
|===
