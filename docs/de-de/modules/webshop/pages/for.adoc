== for-Schleife

Eine for-Schleife kann angewendet werden, indem eine Zählervariable und ein Zahlenbereich definiert wird. Alternativ kann eine Iteration über ein xref:webshop:cms-syntax.adoc#grundlagen-typen[Array] vorgenommen werden.
