Mit dem Datenformat *CustomerNewsletter* exportierst du Daten von Kunden, die für den Bezug von Newslettern eingetragen sind. Du erstellst und exportierst das Datenformat im Menü *Daten » Dynamischer Export*. Wie du das Datenformat exportierst, ist auf der Handbuchseite <<daten/export-import/daten-exportieren#, Daten exportieren>> beschrieben.

Achte auf die exakte Schreibweise der Datenfelder, da die Daten sonst nicht verarbeitet werden können.

[TIP]
.Export von Daten
====
Daten werden nicht aus allen E-Mail-Ordnern exportiert. Wenn kein Filter in dem Datenformat aktiviert wurde, werden die Daten aus dem ersten gefundenen Ordner im Menü *Einrichtung » CRM » Newsletter » plentymarkets » E-Mails* exportiert.
====
