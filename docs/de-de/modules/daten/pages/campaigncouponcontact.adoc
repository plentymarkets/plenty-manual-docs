= CampaignCouponContact
:description: CampaignCouponContact
:page-index: false
:id: 46JUBLA
:author: team-crm

Das Datenformat *CampaignCouponContact* liefert Informationen über die Einlösung von Codes über die Kontakte. Ein Import ist für dieses Format nicht möglich, nur der Export.

Du exportierst das Datenformat im Menü *Daten » Dynamischer Export*. Wie du das Datenformat exportierst, ist auf der Handbuchseite xref:daten:alte-tools-daten-exportieren.adoc#[Daten exportieren] beschrieben.

Achte auf die exakte Schreibweise der Datenfelder, da die Daten sonst nicht verarbeitet werden können.
