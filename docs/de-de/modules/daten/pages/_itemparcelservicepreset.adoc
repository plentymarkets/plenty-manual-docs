= ItemParcelServicePreset
:index: false

Mit dem Spezialformat **ItemParcelServicePreset** importieren Sie Versandprofile, die beim Artikel im Tab **Global** aktiviert sind. Dieses Datenformat steht Ihnen im Menü **Daten » Dynamischer Import** ausschließlich zum Import zur Verfügung. Wie Sie das Datenformat importieren, ist auf der Handbuchseite xref:daten:daten-importieren.adoc#[Daten importieren] beschrieben. Ein Exportformat erstellen Sie im Menü <<daten/export-import/daten-exportieren#30, Daten » Export » Elastischer Export>>.

Achten Sie auf die exakte Schreibweise der Datenfelder, da die Daten sonst nicht verarbeitet werden können.
