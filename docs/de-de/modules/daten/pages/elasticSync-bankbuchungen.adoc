[cols="1,3"]
|====
|Einstellung |Erläuterung

|*Zahlungs-ID*
|In welcher Spalte deiner CSV-Datei werden die Zahlungs-IDs der Zahlungen aufgelistet? Wähle diese Spalte hier. +
Wenn eine Zahlungs-ID bereits im System existiert, werden die dazugehörigen Daten aktualisiert. Existiert die Zahlungs-ID noch nicht, wird die Zahlung mit den importierten Daten neu angelegt. +
*_Beachte_*, dass bei diesem Import nur Zahlungen importiert werden, bei denen es sich um *Bankbuchungen* handelt. Zahlungen über Zahlungsanbieter wie z.B. PayPal oder Klarna werden nicht importiert.

== Felder für die Zuordnung

include::partial$zuordnung.adoc[]

[#30]
=== Zuordnung

[#40]
==== Buchungsbetrag

icon:warning[role="red"] [red]#Pflichtwert#

Welcher Buchungsbetrag soll der Zahlung zugewiesen werden? Gib den Buchungsbetrag der Zahlung in deine CSV-Datei ein.

*_Standardwert_*: Kein Standardwert

*_Zulässige Importwerte_*: Numerisch

[#50]
==== Währung

Welche Währung soll der Zahlung zugewiesen werden? Gib die Währung der Zahlung in deine CSV-Datei ein.

*_Standardwert_*: Euro

*_Zulässige Importwerte_*: Alphanumerisch

[#60]
==== Buchungstag

Welches Buchungsdatum soll der Zahlung zugewiesen werden? Gib den Buchungstag in deine CSV-Datei ein.

*_Standardwert_*: Der Tag des Imports

*_Zulässige Importwerte_*: Alphanumerisch

[#70]
==== Kontakt-ID

Welcher Kontakt soll der Zahlung zugewiesen werden? Gib die ID dieses Kontakts in deine CSV-Datei ein.

*_Standardwert_*: Kein Standardwert

*_Zulässige Importwerte_*: Numerisch

[#80]
==== Zahlungseigenschaft

Welche Zahlungseigenschaft soll der Zahlung zugewiesen werden? Gib den Wert der ausgewählten Zahlungseigenschaft in deine CSV-Datei ein.

Zur Auswahl als *Wert* der Zahlungseigenschaft stehen dir:

* Verwendungszweck
* Name des Senders
* E-Mail des Senders
* BLZ des Senders
* Kontonummer des Senders
* IBAN des Senders
* BIC des Senders
* Name des Empfängers

*_Standardwert_*: Kein Standardwert

*_Zulässige Importwerte_*: Alphanumerisch

[#90]
==== Soll/Haben

Ist die Bankbuchung eine Soll- oder Haben-Buchung? Gib dies entsprechend in deine CSV-Datei ein.

*_Standardwert_*: Kein Standardwert

*_Zulässige Importwerte_*: Alphanumerisch
