Die Auftragsherkunft zeigt an, auf welchem Verkaufskanal oder Marktplatz ein Auftrag erzeugt wurde.

Für {price-search-engine} ist noch keine Auftragsherkunft in deinem plentymarkets System vorhanden. Du musst die Auftragsherkunft für {price-search-engine} deshalb manuell im Menü *Einrichtung » Aufträge » Auftragsherkunft* erstellen.

Die IDs 1-8 und IDs ab 101 sind für im System gespeicherte Auftragsherkünften reserviert. Für Herkünfte, die du selbst erstellst, stehen IDs im Bereich 9-100 zur Verfügung.

[.instruction]
Auftragsherkunft erstellen:

. Öffne das Menü *Einrichtung » Aufträge » Auftragsherkunft*.
. Klicke auf *Neu* (icon:plus-square[]).  +
*_Hinweis_:* Wenn alle verfügbaren IDs für manuell erstellte Auftragsherkünfte vergeben sind, ist die Schaltfläche *Neu* (icon:plus-square[]) nicht mehr wählbar. +
→ Das Fenster *Neue Herkunft erstellen* wird angezeigt. +
→ Eine freie ID wird automatisch zugewiesen. Diese ID kannst du jetzt ändern. Nachdem du die Herkunft erstellt hast, kann die ID aber nicht mehr geändert werden.
. Gib einen Namen für die *Auftragsherkunft* ein. +
*_Tipp:_* Du kannst zum Beispiel den Namen des Preisportals hier eingeben.
. Aktiviere die Option *Sichtbar in Auswahl* (material:toggle_on[role=skyBlue]).
. Klicke auf *Erstellen*. +
→ Die Herkunft ist aktiviert.
