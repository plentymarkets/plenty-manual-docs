Wenn du im Assistenten die Optionen *Verkaufspreis anlegen*, *Barcode anlegen* und/oder *Eigenschaft anlegen* gewählt hast, erstellt der Assistent diese Eigenschaften, Verkaufspreise und/oder Barcodes automatisch. Diese neu erstellten Datenfelder musst du nun noch mit den Varianten verknüpfen, die zum Marktplatz exportiert werden sollen. Wie das geht, ist im Bereich *Artikel* des Handbuchs beschrieben:

* So verknüpfst du xref:artikel:preise.adoc#240[Verkaufspreise].
* So verknüpfst du xref:artikel:barcodes.adoc#300[Barcodes].
* So verknüpfst du xref:artikel:eigenschaften.adoc#1420[Eigenschaften und speicherst einen Eigenschaftswert]. +
*_Tipp:_* Eigenschaften, die über den Assistenten erstellt werden, werden automatisch im Menü *Einrichtung » Einstellungen » Eigenschaften » Gruppen* zu der Gruppe *{property-group}* hinzugefügt. Öffne diese Gruppe um zu sehen, welche neuen Eigenschaften verknüpft werden müssen.
