The data format **OrderItems** allows you to export and to edit existing order item positions as well as to add new positions to an order. Create and export the data format in the **Data » Dynamic export** menu. The <<data/export-import/exporting-data#, Exporting data>> page of the manual describes how you export the data format.

Pay close attention to the exact spelling of the data fields. The data cannot be processed if the data fields are spelled incorrectly.
