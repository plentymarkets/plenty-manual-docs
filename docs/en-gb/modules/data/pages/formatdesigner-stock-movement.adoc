== StockMovements
<<table-fields-StockMovements>> lists the fields available for exporting data about stock movements.

[[table-fields-StockMovements]]
.Fields for exporting stock movements
[cols="1,3"]
|====
|Field |Explanation

|attributeValues
|Attribute values

|batch
|xref:stock-management:new-incoming-items.adoc#810[Batch]

|bestBeforeDate
|xref:stock-management:new-incoming-items.adoc#810[Best before date]

|createdAt
|xref:stock-management:managing-stocks.adoc#1100[Timestamp of the stock movement], as it is displayed in the variation's *Stock movement* tab.

|id
|Process ID of the stock movement. You can see the process ID in the data set of a variation in the *Stock movements* tab of the variation.

|itemId
|Item ID

|processRowId
|Source ID of the stock movement. The source can be an order or incoming items. Thus, the source ID can be an order ID or the ID of incoming items.

|processRowType
|ID of the source type of a stock movement. The source type refers to the menu or action that triggered the stock movement.

|quantity
|Quantity of items included in the stock movement. The leading sign shows if stock was booked in or out. +
-: Stock was booked out +
+: Stock was booked in

|reason
|ID of the stock movement reason

|reasonString
|Name of the stock movement reason

|storageLocationName
|Storage location name

|warehouseId
|ID of the warehouse that the stock was booked into our out from.

|warehouseName
|Name of the warehouse that the stock was booked into our out from.
|====

== Variation
<<table-fields-Variation>> lists the fields available for exporting variation data.

[[table-fields-Variation]]
.Fields for exporting variation data
[cols="1,3"]
|====
|Field |Explanation

|hasCalculatedBundleMovingAveragePrice
|Indicates whether the moving average purchase price (net) is xref:item:combining-products.adoc#2500[automatically calculated based on the individual components]. +
1 = Automatic calculation is activated. +
Empty field = Automatic calculation is not activated.

|hasCalculatedBundleNetWeight
|Indicates whether the net weight is xref:item:combining-products.adoc#2500[automatically calculated based on the individual components]. +
1 = Automatic calculation is activated. +
Empty field = Automatic calculation is not activated.

|hasCalculatedBundlePurchasePrice
|Indicates whether the net purchase price is xref:item:combining-products.adoc#2500[automatically calculated based on the individual components]. +
1 = Automatic calculation is activated. +
Empty field = Automatic calculation is not activated.

|hasCalculatedBundleWeight
|Indicates whether the gross weight is xref:item:combining-products.adoc#2500[automatically calculated based on the individual components]. +
1 = Automatic calculation is activated. +
Empty field = Automatic calculation is not activated.

|id
|ID of the variation included in the stock movement

|number
|Variation number

|variationTag
|xref:item:flags.adoc#400[Variation tags]
|====
