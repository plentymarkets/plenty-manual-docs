The data format *ItemListingMarketHistory* is used to export active listings. Create and export the data format in the *Data Â» Dynamic export* menu. The <<data/export-import/exporting-data#, Exporting data>> page of the manual describes how you export the data format.

Pay close attention to the exact spelling of the data fields. The data cannot be processed if the data fields are spelled incorrectly.

Table 1 explains the data fields for the *ItemListingMarketHistory* data format in alphabetical order as well as filter options that you set in the *Filter* tab of the data format.
