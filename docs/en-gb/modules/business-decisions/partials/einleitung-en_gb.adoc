Use the {dataFormat} data format creator to export {data} and save this data in the cloud.
Depending on the plentyBI edition that has been booked, the raw data file is stored in the cloud for 6 to 48 months. Therefore, the data must be processed before this period of time ends. More detailed information on the storage periods are available on the <<business-decisions/plenty-bi/reports/manage-reports#_data_format_creator#, Managing reports>> page of the manual.
