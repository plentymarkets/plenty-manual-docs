= Messages
:keywords: messages
:author: team-crm


:key-figure: Messages

include::partial$key-figure-introduction.adoc[]

plentymarkets calculates the key figure *Messages* based on the number of messages or the sum of minutes that were saved for the messages.

By selecting a xref:business-decisions:messages.adoc#100[calculation type], you decide whether to calculate with the number of messages or the sum of minutes. In addition, select a xref:business-decisions:messages.adoc#200[dimension] and xref:business-decisions:messages.adoc#300[filters] to specify the criteria for calculating the messages.
