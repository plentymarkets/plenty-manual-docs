ifdef::manual[]
Enter a number for the variation.

*_Recommendation_*: If you leave this field blank, then plentymarkets will automatically assign the <<item/settings/variation-numbers#, next available variation number>>.
The number will be assigned according to the pattern saved in the menu *Setup » Item » Settings*.
endif::manual[]

ifdef::import[]
Enter a number for the variation into the CSV file.
<<item/import-export-create/import#400, See the field in a practical example>>.
If you leave this field blank, then plentymarkets will automatically assign the <<item/settings/variation-numbers#, next available variation number>>.
The number will be assigned according to the pattern saved in the menu *Setup » Item » Settings*.

*_Default value_*: No default value

*_Permitted import values_*: Alpha-numeric

You can find the result of the import in the back end menu: <<item/managing-items#190, Item » Edit item » [Open variation] » Tab: Settings » Area: Basic settings » Entry field: Variation No.>>
endif::import[]

ifdef::export[]
The variation’s number.

Corresponds to the option in the menu: <<item/managing-items#190, Item » Edit item » [Open variation] » Tab: Settings » Area: Basic settings » Entry field: Variation No.>>
endif::export[]
