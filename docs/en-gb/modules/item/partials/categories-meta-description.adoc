ifdef::manual[]
Enter a meta description (max. 350 characters).
The meta description is analysed by search engines. As such, it is especially important for the homepage.
The meta description should contain the most important content in one sentence.
endif::manual[]

ifdef::import[]
Enter a meta description (max. 350 characters) into the CSV file.
The meta description is analysed by search engines. As such, it is especially important for the homepage.
The meta description should contain the most important content in one sentence.

*_Default value_*: No default value

*_Permitted import values_*: Alpha-numeric

[TIP]
Also use the drop-down lists to select the language and the webstore.

You can find the result of the import in the back end menu: <<item/settings/categories#intable-meta-description, Item » Category » Tab: Settings » Entry field: Meta description>>
endif::import[]

endif::export,catalogue[]
