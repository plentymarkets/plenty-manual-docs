ifdef::manual[]
Enter a meta title.
The meta title is displayed in the tab title of the browser and in search engine results.
If you leave this field blank, the category name is used as the title.

*_SEO tip_*: Pay attention to the length of the title.
Longer titles are cut off in the tabs.
Currently recommended: max. 50 characters
endif::manual[]

ifdef::import[]
Enter a meta title into the CSV file.
The meta title is displayed in the tab title of the browser and in search engine results.

*_Default value_*: No default value

*_Permitted import values_*: Alpha-numeric

[TIP]
Also use the drop-down lists to select the language and the webstore.

You can find the result of the import in the back end menu: <<item/settings/categories#intable-meta-data, Item » Category » Tab: Settings » Entry field: Meta title>>

*_SEO tip_*: Pay attention to the length of the title.
Longer titles are cut off in the tabs.
Currently recommended: max. 50 characters
endif::import[]

ifdef::export,catalogue[]
The category’s meta title.
The meta title is displayed in the tab title of the browser and in search engine results.

Corresponds to the option in the menu: <<item/settings/categories#700, Item » Category » [Open category] » Tab: Settings » Entry field: Meta title>>
endif::export,catalogue[]
