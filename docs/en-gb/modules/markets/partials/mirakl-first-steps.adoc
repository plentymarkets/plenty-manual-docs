[#register]
== Registering with {market}

In order to set up {market} in plentymarkets, you need your {market} access data which consist of your user name and password. You will receive this data from {market}. First, you have to create a user account for link:{registration-url}[{market}^]{nbsp}icon:external-link[].

[#about]
== About {market}

In order to sell your items on the {type-of-market} {market} with plentymarkets, you need the *Mirakl Connector* plugin. The *Mirakl Connector* plugin connects {market} with your plentymarkets system. This enables {market} and plentymarkets to automatically exchange item and order data and you can offer your items on {market}.

In order to be able to sell your items on {market}, you therefore have to <<#downloading-plugins, install the *Mirakl Connector* plugin>> first.

[#download-plugins]
== Downloading the plugins Mirakl Connector and {market}

You need a plugin in order to be able to offer your items on {market} with plentymarkets. Therefore, download the {market} plugin in the link:{marketplace-url}[plentyMarketplace^]{nbsp}icon:external-link[] first.

{market} is connected with plentymarkets via the platform Mirakl. To use the {market} plugin, you also need the *Mirakl Connector* plugin. You can download the *Mirakl Conncetor* plugin in the link:https://marketplace.plentymarkets.com/en/mirakl_6917[plentyMarketplace^]{nbsp}icon:external-link[].
