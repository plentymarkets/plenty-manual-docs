On this page you will learn how to download and install the plugins *Mirakl* and *{market}*. Mirakl is the platform via which {market} is connected.

[#register]
== Registering with {market}

You need your {market} login details consisting of the user name and password. You will receive this data from {market}. First, you have to create a user account for link:{registration-url}[{market}^]{nbsp}icon:external-link[].

[#about]
== About {market}

The *Mirakl Connector* plugin connects the {type-of-market} {market} with your plentymarkets system. The plugin enables you to automatically exchange item and order data between plentymarkets and the market {market}. This way, you can list your items on {market}.

[#download-plugins]
== Downloading the plugins Mirakl Connector and {market}

You need a plugin to link plentymarkets to {market}. In order to use the {market} plugin, download it in the link:{marketplace-url}[plentyMarketplace^]{nbsp}icon:external-link[].

To use the {market} plugin, you also need the *Mirakl Connector* plugin. Mirakl is also available in the link:https://marketplace.plentymarkets.com/en/mirakl_6917[plentyMarketplace^]{nbsp}icon:external-link[].
