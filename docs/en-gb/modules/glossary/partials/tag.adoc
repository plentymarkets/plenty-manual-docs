[#tag]
Tags make it possible to emphasise certain data records. After tags have been assigned, data records can be filtered according to tags, allowing for better overview. Tags can be applied to individual xref:item:managing-items.adoc#268[variations], xref:item:categories.adoc#700[categories], xref:crm:preparatory-settings.adoc#create-tags[contacts], xref:crm:messenger-testphase.adoc#create-message[messages in the Messenger], xref:crm:using-the-ticket-system.adoc#1600[tickets] and xref:welcome:customise-system.adoc#190[tasks].
