[#box-storage-rack]
Box storage rack:: Box storage racks in plentymarkets are a kind of interim storage for multi order picking and are only available in xref:automation:procedures.adoc#[processes]. Thus, when picking stock, the items are not sorted immediately, but gathered in a box storage rack. +
* For further information, refer to the xref:app:mobile-box-picking.adoc#[Mobile box picking] page.
